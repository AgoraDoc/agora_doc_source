IAudioDeviceCollection

IAudioDeviceCollection 类。你可以通过该接口类获取音频设备相关的信息。

getCount

获取播放或音频采集设备数量。

virtual int getCount() = 0;

如果先调用 enumeratePlaybackDevices再调用该方法，返回播放设备的数量。 如果先调用 enumerateRecordingDevices再调用该方法，则返回音频采集设备的数量。

返回值

语音设备数目。

getDevice

获取指定 index 的设备信息。

virtual int getDevice(int index, char deviceName[MAX_DEVICE_ID_LENGTH], char deviceId[MAX_DEVICE_ID_LENGTH]) = 0;

参数

index
输入参数，指定想查询的设备信息。
deviceName
输出参数，设备名称。最大长度为 MAX_DEVICE_ID_LENGTH。
deviceId
输出参数，设备 ID。最大长度为 MAX_DEVICE_ID_LENGTH。

返回值

  • 0: 方法调用成功。
  • < 0: 方法调用失败。

setDevice

指定音频设备。

virtual int setDevice(const char deviceId[MAX_DEVICE_ID_LENGTH]) = 0;

参数

deviceId
设备 ID。最大长度为 MAX_DEVICE_ID_LENGTH。

返回值

  • 0: 方法调用成功。
  • < 0: 方法调用失败。

setApplicationVolume

设置 App 音量。

virtual int setApplicationVolume(int volume) = 0;

参数

volume
音量值，范围 [0,255]。

返回值

  • 0: 方法调用成功。
  • < 0: 方法调用失败。

getApplicationVolume

获取 App 的当前音量。

virtual int getApplicationVolume(int& volume) = 0;

参数

volume
音量值，范围 [0,255]。

返回值

  • 0: 方法调用成功。
  • < 0: 方法调用失败。

setApplicationMute

设置 App 静音。

virtual int getApplicationVolume(int& volume) = 0;

参数

mute
是否设置 App 静音。
  • true: 设置 App 为静音。
  • false: 不设置 App 为静音。

返回值

  • 0: 方法调用成功。
  • < 0: 方法调用失败。

isApplicationMute

获取 App 当前的静音状态。

virtual int isApplicationMute(bool& mute) = 0;

参数

mute
App 当前静音状态指针：
  • true: App 为静音状态。
  • false: App 为非静音状态。

返回值

  • 0: 方法调用成功。
  • < 0: 方法调用失败。

release

释放 IAudioDeviceCollection 对象占用的所有资源。

virtual void release() = 0;
