API Overview

声网通过全球部署的虚拟网络，提供可以灵活搭配的 API 组合，为移动端到移动端以及移动端到 Web 端提供质量可靠的实时音视频通信。

注：
  • 自 v2.3.3 开始，Agora C++ API Reference for All Platforms 可以在 Dash 平台下载，详见 FAQ。
  • 自 v3.0.0 开始，Agora 废弃 RtcEngineParameters 类。该类下所有接口均被 IRtcEngine 类覆盖，请使用 IRtcEngine 类。
  • 在调用 API 过程中，SDK 可能会返回错误码和警告码，详见错误码和警告码。
  • IRtcEngine 类包含应用程序调用的主要方法。
  • IRtcEngineEventHandler 类用于向应用程序发送回调通知。
  • IChannel 类在指定频道中实现实时音视频功能。通过创建多个 IChannel 对象，用户可以同时加入多个频道。
  • IChannelEventHandler 类监听和报告指定频道的事件和数据。
  • IAudioDeviceManager 类提供用于测试音频设备的相关接口。
  • IVideoDeviceManager 类提供用于测试视频设备的相关接口。
