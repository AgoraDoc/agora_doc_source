onClientRoleChanged

直播场景下用户角色已切换回调。

virtual void agora::rtc::IRtcEngineEventHandler::onClientRoleChanged(CLIENT_ROLE_TYPE oldRole, CLIENT_ROLE_TYPE newRole)

用户角色切换后会触发该回调，如从观众切换为主播，反之亦然。

该回调由本地用户在加入频道后调用 setClientRole 改变用户角色触发的。
