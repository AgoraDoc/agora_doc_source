LiveTranscoding

管理旁路推流配置的类。

agora::rtc::LiveTranscoding::LiveTranscoding()

属性

名称 类型 描述
width Integer

推流视频的总宽度，默认值 360，单位为像素。

  • 如果推视频流，width 值不得低于 64，否则 Agora 会调整为 64。
  • 如果推音频流，请将 width 和 height 设为 0。
videoCodecProfile VIDEO_CODEC_PROFILE_TYPE
用于旁路推流的输出视频的编码规格。
注： 如果你将这个参数设为其他值，Agora 会将其设为默认值 100。
transcodingUsers TranscodingUser

管理参与旁路直播的音视频转码合图的用户。最多支持 17 个用户。
