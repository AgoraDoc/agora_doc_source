TranscodingUser

管理参与旁路直播的音视频转码的用户。

属性

uid
类型：uid_t
旁路主播的用户 ID。
x
类型：Integer
屏幕里该区域相对左上角的横坐标绝对值 (pixel)。取值范围为转码配置（LiveTranscoding）定义中设置的 [0,width]。
