Agora Java API Reference for Android

声网通过全球部署的虚拟网络，提供可以灵活搭配的 API 组合，提供质量可靠的实时音视频通信。

  • RtcEngine 类包含应用程序调用的主要方法。

  • IRtcEngineEventHandler 类用于向应用程序发送回调通知。

自 v2.3.3 开始，Agora Java API Reference for Android 可以在 Dash 平台下载，详见 FAQ。
