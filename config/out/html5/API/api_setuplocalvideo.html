setupLocalVideo

    初始化本地视图。

    原型

    virtual int agora::rtc::IRtcEngine::setupLocalVideo (const VideoCanvas & canvas)

    详细描述

    该方法初始化本地视图并设置本地用户视频显示信息，只影响本地用户看到的视频画面，不影响本地发布视频。调用该方法绑定本地视频流的显示视窗(view)，并设置本地用户视图的渲染模式和镜像模式。

    在 App 开发中，通常在初始化后调用该方法进行本地视频设置，然后再加入频道。退出频道后，绑定仍然有效，如果需要解除绑定，可以指定空(NULL)View 调用该方法。

    注：
    • 该方法在加入频道前后都能调用。
    • 如果你希望在通话中更新本地用户视图的渲染或镜像模式，请使用 方法。

    参数

    canvas
    视频画布信息: VideoCanvas 。

    返回值

    • 0：方法调用成功。
    • <0：方法调用失败。
